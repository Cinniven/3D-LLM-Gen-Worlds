# 3D-LLM-Gen-Worlds
Welcome to group 701 semester project for semester 7
____________________________________________________

How to run the project via Unity:
1. Unpack this Zip
2. Make sure to open it in Unity version 6000.0.58f1
3. Open Unity Project
4. Open The Scene Called "ChunkAITest"
5. Click Managers under the Hierarchy
6. Find the child object under the managers called "LLMParent"
7. Find the child object called "LLM"
8. Download a Model by inserting one of our Model Links or Download one from a package

Model Download Links:
https://huggingface.co/GrimTale/Qwen3-8B-ChunkModels/resolve/main/Model%20Saved/Qwen3-8B-100.gguf?download=true

https://huggingface.co/GrimTale/Qwen3-8B-ChunkModels/resolve/main/Model%20Saved/Qwen3-8B-200.gguf?download=true

9. Run the Scene 
10. Use the object child "MyLLM" to generate LLM Chunk
